## Comprehension Question

Graceful degradation and service workers are related because service workers help a website keep working when network access is limited or unavailable. Graceful degradation means the app should still provide a usable experience even if something fails, like the internet connection. In this lab, the service worker caches the recipe app’s files and recipe data, so when the user goes offline, the app can still load from the cache instead of completely breaking.
